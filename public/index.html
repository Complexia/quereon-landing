how it works

quereon uses gpt-3 to interpret natural language and context and transform it into code that will execute on your data to yield desired results

Natural language

Quereon processes natural language queries using GPT-3 to understand the desired action and context, and generates code to transform data in meaningful way.

‍

Learn more

Tensor search

By storing data queries as vectors, and then performing a tensor search using the natural language input, Quereon is able to pre-fill the generated code to a certain level of accuracy, and then the GPT-3 model follows on by patching up the gaps and making it even more accurate. This results in cheaper and faster queries.
